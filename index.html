⚡ 충전 & 조각 적립

1) 화면을 끄거나, 앱을 꺼도 조각이 쌓이나요?

네, 그럼요! 충전기를 꽂은 상태라면 다른 앱을 이용하시거나 화면을 꺼도 조각이 자동으로 적립됩니다.

2) 충전 케이블 말고 무선 충전도 되나요?

그럼요! 무선 충전, 보조배터리 어떤 수단으로든 핸드폰이 충전되고 있을 때는 조각이 적립됩니다.

3) 인터넷 연결을 꺼도 조각이 쌓이나요?

아니요! 데이터 및 와이파이 연결 환경에서만 조각이 정상적으로 적립됩니다.

4) 충전 중인데 조각이 안 쌓이는 이유가 뭔가요?

충전기 상태, 데이터 환경이 정상적인데도 조각이 쌓이지 않는다면, 앱 내 [설정] → [문의하기]를 통해 남겨주시면 최대한 빠르게 확인 후 답변드릴게요.

5) 친구초대 보상은 어떻게 받나요?

최초 로그인 시 추천인코드 입력 페이지가 노출됩니다. 해당 화면에서 추천인코드를 입력 후 완료 버튼을 눌러 로그인이 완료되는 즉시, 친구와 나에게 각각 500캐시가 지급됩니다.

최초 로그인 후에는 코드 추가 등록이 어려운 점 참고 부탁드립니다.

주의: 추천인코드 입력 페이지를 한 번 나가기 하거나 앱을 종료하면, 다시 코드 입력 페이지는 노출되지 않습니다. 반드시 해당 페이지에서 추천인코드 입력을 완료하세요.

💎 조각 & 원석 & 정제

1) 조각은 하루에 몇 개까지 모을 수 있나요?

충전 중 10분마다 조각 1개가 자동으로 적립됩니다. 하루 최대 100개의 조각을 모을 수 있습니다.

2) 원석은 어떻게 만드나요?

조각 10개가 모이면 원석 1개를 제작할 수 있습니다. 홈 화면의 [원석 제작] 버튼을 눌러주세요.

잠깐! 조각이 100개를 넘어가면 추가 적립이 되지 않습니다. 원석 제작을 통해 조각을 소모해주세요.

3) 정제는 어떻게 하나요?

원석이 1개 이상 있으면 정제할 수 있습니다. 정제 화면에서 [일반 정제] 또는 [프리미엄 정제]를 선택하세요.

  • 일반 정제: 무료로 정제하여 캐시를 획득합니다.
  • 프리미엄 정제: 광고 시청 후 더 높은 등급의 보상에 도전할 수 있습니다.

4) 원석이 있는데 정제가 안 됩니다. 어떻게 해야 할까요?

정제 화면에서 [잠시 후 다시 시도하세요] 메시지가 보이시나요? 아래 사항을 확인해주세요.

  • 광고 차단 앱이 설치되어 있진 않나요?
  • 휴대폰 설정 → 프라이빗 DNS 설정에서 '자동'이 아닌 다른 설정으로 되어 있진 않나요?
  • 프리미엄 정제는 광고 시청 후 진행되는 방식이라, 위 설정에서는 참여가 어렵습니다.
  • 연속적으로 많은 광고 시청 시 광고사에서 일시적으로 광고 노출을 제한할 수 있어요. 시간을 조금 두었다가 다시 시도해보세요.

💳 출금 & 계정

1) 출금은 얼마부터 가능한가요?

3,000원 이상부터 출금 가능합니다. (출금 시 건당 500원 송금 수수료 발생)

  • 예: 3,000원 출금 시 입금액 2,500원
  • 3,000원 / 5,000원 / 10,000원 / 20,000원 / 30,000원 단위로 선택 가능
  • 보유 캐시보다 큰 금액은 선택할 수 없습니다.

앱 내에서 사용되는 '캐시'는 서비스 상의 통화로, 실제 현금 '원'과 동일한 금액으로 사용되지 않는 점 참고 바랍니다. (예: 현금 3,000원 출금에 필요한 캐시는 5,331캐시)

2) 출금은 얼마나 걸리나요?

신청서에 올바른 정보를 기재해주신 경우, 영업일 기준 7일 이내 출금이 진행됩니다.

3) 출금 한도가 있나요?

일일 최대 30,000원, 월간 최대 50,000원까지 출금 가능합니다.

4) 계좌번호를 잘못 입력했어요.

출금 신청 시 입력한 계좌번호에 대한 책임은 사용자에게 있습니다. 잘못된 계좌로 이체된 건에 대해서는 책임지지 않는 점 양해 부탁드립니다.

5) 계정 변경이나 휴대폰 교체 시, 기존 데이터는 어떻게 되나요?

  • 비회원(게스트)으로 이용 중인 경우, 데이터 이전이 어려울 수 있습니다. 안전한 데이터 저장을 위해 소셜 로그인 후 이용해주세요.
  • 새로운 기기에서도 동일 계정으로 로그인하시면 데이터가 정상적으로 이전됩니다.

🌎 기타

1) 해외에서도 사용할 수 있나요?

그럼요! 해외에서도 충전하는 시간 동안 조각이 적립됩니다.

2) 알림이 오지 않아요.

휴대폰 설정 → 앱 → 캐시드릴 → 알림에서 알림이 허용되어 있는지 확인해주세요. 또한 배터리 최적화에서 캐시드릴이 제외되어 있는지도 확인해주세요.
